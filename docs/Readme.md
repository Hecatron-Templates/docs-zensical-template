# Readme

Zensical based documentation template

  * https://zensical.org

## Commands

| Command                  | Action                                            |
| :----------------------- | :------------------------------------------------ |
| `virtenv/setup_venv.sh`  | Setup the python virtual env (bash)               |
| `virtenv/setup_venv.ps1` | Setup the python virtual env (powershell)         |
| `build.py build`         | Build the site                                    |
| `build.py clean`         | clean out the site / build directory              |
| `build.py serve`         | Serve / preview the site on http://localhost:8000 |
